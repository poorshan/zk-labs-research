# 关于 ZK-Labs Research

## 我们是谁

ZK-Labs Research 是 **ZK Capital**（Singapore）旗下的独立加密研究机构。

我们不卖数据，不接付费推广，不替任何项目方站台。每一篇报告的唯一客户，是我们自己的交易决策——以及未来选择信任我们判断的合作伙伴。

在信息过载的加密世界里，我们专注于一个被大多数人忽视的维度：**波动率**。我们认为，理解波动率定价错误的深度，比预测价格方向更能持续创造 alpha。

---

## 研究方法

**四条铁律：**

1. **无来源，不落笔** —— 所有数据论断必须标注可验证来源（DeFiLlama、CoinGecko、Deribit、Glassnode 等），不依赖记忆或二手转述
2. **时间戳不可篡改** —— 每一篇研报通过 GitHub commit 发布，commit hash 就是审计线索。不删不改，错了公开勘误
3. **数据实时验证** —— 关键数字在撰写时通过 API 实时抓取，不引用过期快照
4. **批判性独立** —— 不因为持仓而美化，不因为踏空而贬低。观点可能有错，立场不能有偏

## 研究领域

| 领域 | 具体方向 |
|------|----------|
| **加密期权** | ETH/BTC 波动率曲面、期限结构、偏度异常、策略回测 |
| **波动率宏观** | VIX 择时、跨资产波动率传导、波动率风险溢价 |
| **协议基本面** | 稳定币、L1/L2、MEV、流动性质押等赛道的深度分析 |
| **量化交易系统** | 期权策略引擎、波动率定价模型、风控框架 |

## 团队

ZK-Labs Research 由 **金戊乾坤2号** 投研团队运营。

我们采用「人机协作」模式：量化系统负责数据采集、异常检测和策略信号生成，人工负责方向性判断、主题研究和最终决策。团队的核心竞争力不在算力，而在**将波动率判断和仓位纪律编码为可执行系统的能力**。

## Track Record 承诺

ZK-Labs Research 的每一篇研报都是一个不可篡改的公开记录。

我们不要求你相信我们的判断——我们要求你验证我们的记录。所有历史报告在 [GitHub 仓库](https://github.com/poorshan/zk-labs-research) 中完整保留，每篇报告的发布时间由 git commit 时间戳锁定。一年后、三年后，你仍然可以回溯验证：当时我们看到了什么，我们判断对了什么，我们错过了什么。

这是 ZK Capital 最重要的资产。对未来的合作伙伴而言，这比任何回测曲线都更有说服力。
